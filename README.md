## Running Project
### `.env`
```
REACT_APP_FEEDBACK_PROJ_ID = "XXX"
REACT_APP_FEEDBACK_PROJ_NAME = "XXX"
```
### `npm run start`

Runs the app in the development mode.<br /> Open
[http://localhost:3000](http://localhost:3000) to view it in the browser.
